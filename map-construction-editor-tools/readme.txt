Small tool to modify objects while working with
SA-MP: Map Construction/SAMP-MC/samp-map-editor/editor.exe/JernejL's map editor
or anything that generates CreateObject calls.

features
--------
Bulk edit ID/X/Y/Z:
 change all to specific value
 change all to minimum value
 change all to maximum value
 change all to average value
 distribute all uniformly from min to max

Usual worflow
-------------
Create some objects in SAMP-MC
Click 'Show Code'
Copy the code
Open tool/copy code into tool
Select the lines with the objects that you want to edit
Click some bulk edit button
Do edits, click 'Ok'
Click the copy button
Go back to SAMP-MC
Click 'Clear ALL'
Click 'Import Paste'
